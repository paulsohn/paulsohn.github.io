# 08/23
## Jekyll 설치

```
$ sudo apt install ruby
$ sudo apt install gem
$ gem install jekyll bundler
```

좀 시간이 걸린다.

```
$ jekyll new --skip-bundle .
```

이제 게시글은 [Prose](https://prose.io)에서 올릴 수 있을듯.

```
$ bundle install
$ bundle exec jekyll serve
```
